Программа проверяет умножение в двоичных кодах.
Предусмотренно два варианта стенда: для версии Quartus 18.1

Инструкция:
1)Копируете фаил с программой (*.vhd) в папку со стендом.   
2)Открываете проект, открываете в нем файл, добавляете его в проект 
Project=> Add Current File to Project   
3)Создаете из программы блок-элемент
File=> Create/Update=> Create Symbol Files for Current File
4)Вытаскиваете элемент на блок схему вместо элемента Multiply
два раза щелкнуть на пустом месте на блок схеме, раскрыть Project и выбрать ваш элемент
5)Убедиться на схеме в соответствии n-разрядности операций и mode - формату чисел.
5)Скомпилировать проект
6)В вектроной диаграмме (Waveform.vwf) запустить функциональное моделирование.
7)В отчете симуляции:

Сигнал x - множимое.
Сигнал y - множитель.
Сигнал rez_test - произведение.
Сигнал rez_true - истинное произведение.
okey - истина, значит значение совпало, ложь - значения расходятся*

Если временная таблица Waveform повреждена, возможно создать новый файл временных таблиц.
После загрузки входных и выходных пинов следует на первом такте выставить сигнал reset в активное состояние.
Интервал тактовых сигналов рекомендуется выставить в 3 наносекунды


Существуют 6 режимов.
Для режимов с целыми числами 7-ый разряд знаковый, 6-0 числовые.
Для режимов с дробными числами 7-ый разряд знаковый, 6-1 числовые, 0-ой разряд равен нулю (или единице, в зависимомти от кода).

Режим устанавливается сигналом "Mode":

0   Прямой код, целые числа.
1   Прямой код, дробные числа.
2   Обратный код, целые числа.
3   Обратный код, дробные числа.
4   Дополнительный код, целые числа.
5   Дополнительный код, дробные числа.
